# Get-CoffeeTracker

## SYNOPSIS
Returns the Path of the current CoffeeTracker File.

## SYNTAX

```
Get-CoffeeTracker [<CommonParameters>]
```

## DESCRIPTION
Returns the Path of the current CoffeeTracker File.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-CoffeeTracker
```

The only thing it does.
